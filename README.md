## 🚀Install

### docker-compose

```bash
cd oneterm
cp cmd/api/confTemplate.yaml cmd/api/conf.yaml # edit your config
cp cmd/ssh/confTemplate.yaml cmd/ssh/conf.yaml # edit your config
docker compose up -d
```
